# Premiers pas

Ce guide vous mène de l'installation à votre premier export en
30 minutes environ.

1. [Installation](installation.md) — uv, dépendances système
2. [Configuration](configuration.md) — base de données et chemins
   de scans
3. [Premier import](premier-import.md) — importer un fonds depuis
   un profil YAML
4. [Workflow type](workflow-type.md) — du dépôt physique à la
   publication Nakala

Vous travaillez avec une équipe ? Une fois l'outil installé sur
votre machine, vos collègues peuvent suivre le même guide. Une
configuration partagée (base sur serveur, scans sur partage) est
prévue mais sort du périmètre de ce guide.
