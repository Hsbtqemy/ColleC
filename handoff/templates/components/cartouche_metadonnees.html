{# --------------------------------------------------------------------------- cartouche_metadonnees.html — bloc métadonnées style Zotero --------------------------------------------------------------------------- Composé de plusieurs macros : cartouche(item) → entrée principale ; rend toutes les sections d'un item. Attendu : `item` expose .identification = [(label, render_macro_call_str)] ou un dict structuré .identifiants_externes / .champs_personnalises / .description ... En pratique on appellera plutôt les macros de section directement depuis le template de page `pages/item.html` pour garder la main sur les valeurs spéciales. section(titre, info=None, default_open=True) (block) ligne(label) (block) valeur_mono(text) (helper) valeur_doi(href, text) (helper) valeur_url(href) (helper) valeur_date_incertaine(text) (helper) valeur_non_renseigne() (helper) liste_auteurs(valeurs) (helper) tags_sujets(valeurs) (helper) L'édition inline (v0.7) cible `[data-edit-field]` sur chaque conteneur de valeur — déjà posé dans le markup. --------------------------------------------------------------------------- #} {%- macro section(titre, info=None, default_open=True, anchor=None) -%}

{{ caller() }}
{%- endmacro %} {%- macro ligne(label, field=None) -%}
{{ label }}

{%- endmacro %} {%- macro valeur_mono(text) -%} {{ text }} {%- endmacro %} {%- macro valeur_doi(href, text=None) -%} {{ text or href }} {%- endmacro %} {%- macro valeur_url(href) -%} {{ href }} {%- endmacro %} {%- macro valeur_date_incertaine(text) -%} {{ text }} {%- endmacro %} {%- macro valeur_non_renseigne() -%} non renseigné {%- endmacro %} {%- macro liste_auteurs(valeurs) -%}
{%- for v in valeurs %}

{%- endfor %}
{%- endmacro %} {%- macro tags_sujets(valeurs) -%}
